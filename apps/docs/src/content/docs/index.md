## What is AEOrank?

AEOrank is an open-source CLI tool that scans your website and measures how visible it is to AI engines like ChatGPT, Perplexity, and Claude. It scores your site across 12 dimensions and generates the 8 files that AI engines look for.

## Get Started

Run a single command to scan any website:

```bash
npx aeorank-cli scan https://your-site.com
```

You'll get an AEO score from 0-100 and 8 generated files ready to deploy. [Read the quick start guide](/getting-started/) to learn more.
