Generate AEO (AI Engine Optimization) files for your WordPress site.

== Description ==

AEOrank generates the 8 files that help AI engines discover, understand, and cite your content:

* llms.txt — Summary for LLM crawlers
* llms-full.txt — Extended context
* CLAUDE.md — Repository context for AI assistants
* schema.json — JSON-LD structured data
* robots-patch.txt — AI bot access rules
* faq-blocks.html — FAQ with schema markup
* citation-anchors.html — Citation anchor examples
* sitemap-ai.xml — AI-optimized sitemap

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate through the Plugins menu
3. Go to Settings > AEOrank to configure
