2. In `packages/core/src/scorer/dimensions.ts`, add 3 new scorer functions:

   **scoreCrossPageDuplication** (weight: "low"):
   - Collect all paragraphs from all pages, normalize (lowercase, strip extra whitespace)
   - Build a Map<string, number> counting how many DIFFERENT pages each normalized paragraph appears on
   - Count paragraphs appearing on 2+ pages
   - Calculate ratio: duplicated paragraphs / total unique paragraphs
   - Score: inverted — 0 ratio = 10, <0.05 = 8, <0.1 = 5, <0.2 = 3, else 0
   - If fewer than 2 pages, return score 10 with hint "Single page — cross-page duplication not applicable"
   - Hint: "Remove identical content blocks that appear across multiple pages"

   **scoreEvidencePackaging** (weight: "low"):
   - For each page, check sentences for evidence markers:
     - Attribution: `/according to/i`, `/\bsource[s]?:/i`, `/\bcited?\b/i`, `/\breference[s]?:/i`
     - Inline citations: `/\[\d+\]/`, `/\(.*?\d{4}.*?\)/` (parenthetical year citations)
     - Sources sections: Check if any heading text matches `/source|reference|bibliograph|citation/i`
   - Count pages with at least 2 evidence markers
   - Score: >50% pages = 10, >30% = 7, >15% = 4, >0% = 2, else 0
   - Hint: "Add inline citations, attribution phrases, and source references to support claims"

   **scoreCitationReadyWriting** (weight: "low"):
   - For each page, analyze sentences for citation-readiness:
     - Definition patterns: `/^[A-Z][^.]+\s+(is|are|refers to|means|describes|defines)\s+/`
     - Self-contained statements: sentences between 40-200 chars that start with a capital letter and end with a period (not a question or exclamation)
     - Single-claim sentences: sentences with no "and" or "but" conjunctions connecting independent clauses (simple heuristic: no `, and ` or `, but ` pattern)
   - Count citation-ready sentences per page, calculate ratio vs total sentences
   - Average the ratio across pages
   - Score: avg ratio >0.4 = 10, >0.25 = 7, >0.15 = 4, >0.05 = 2, else 0
   - Hint: "Write self-contained definition sentences and single-claim statements that AI can quote directly"
