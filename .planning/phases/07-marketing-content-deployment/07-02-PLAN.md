  2. Click "Draft a new release"
  3. Create tag `v1.0.0` targeting the main branch
  4. Set release title: "AEOrank Action v1.0.0"
  5. Check "Publish this Action to the GitHub Marketplace"
  6. Fill in Marketplace categories (suggested: "Code quality", "Continuous integration")
  7. Write release notes summarizing action capabilities
