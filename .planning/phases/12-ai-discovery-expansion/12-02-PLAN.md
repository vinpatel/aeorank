    4. Check if homepage has rssFeeds with length > 0
    5. Check if ANY page has rssFeeds
    6. Scoring:
       - Homepage has RSS/Atom feed: score = 10
       - Non-homepage has RSS/Atom feed: score = 4 (feed exists but not discoverable from homepage)
       - No feeds anywhere: score = 0

    6. baseDateScore from dateRatio: >= 0.8 -> 8, >= 0.6 -> 6, >= 0.4 -> 4, >= 0.2 -> 2, > 0 -> 1, else 0
    7. metaBonus: count pages with hasDatePublished / pages.length >= 0.5 -> +2, else 0
    8. score = min(10, baseDateScore + metaBonus)
    9. Hint: "Add <time datetime='...'> elements to show publication and update dates visibly on content pages"
