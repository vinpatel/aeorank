    **scoreContentCannibalization** (Dimension 33, low weight, id "content-cannibalization"):
    1. If pages.length <= 1, return score 10 (no cannibalization possible)
    2. Sort pages by URL for determinism
    3. Normalize titles: lowercase, strip common suffixes (| SiteName, - SiteName)
    4. For each pair of pages, compute title similarity using word overlap (Jaccard):
       - Split titles into word sets
       - Jaccard = |intersection| / |union|
       - If Jaccard > 0.7, count as a cannibalization pair
    5. Also check heading overlap: if two pages share > 50% of h2 headings (by normalized text), count as overlap
    6. cannibalizedPairs / totalPossiblePairs = cannibalRatio
    7. Scoring: ratio === 0 -> 10, ratio < 0.1 -> 8, ratio < 0.2 -> 6, ratio < 0.4 -> 4, ratio < 0.6 -> 2, else -> 0
    8. Hint: "Consolidate pages with overlapping topics to avoid content cannibalization"

    **scorePublishingVelocity** (Dimension 34, low weight, id "publishing-velocity"):
    1. Use meta.sitemapLastmods
    2. If no lastmods or length < 2, return score 0 with hint "Add lastmod dates to sitemap.xml entries"
    3. Parse dates, sort chronologically, filter valid dates only
    4. Compute:
       - recency: most recent date vs now (within 90 days = good)
       - regularity: standard deviation of gaps between consecutive dates (lower = more regular)
       - span: time span from oldest to newest (longer = more established)
    5. recencyScore (0-4): within 30 days = 4, 90 days = 3, 180 days = 2, 365 days = 1, else 0
    6. regularityScore (0-3): stddev of gaps < median gap -> 3, < 2x median -> 2, else 1
    7. spanScore (0-3): span > 180 days = 3, > 90 = 2, > 30 = 1, else 0
    8. score = recencyScore + regularityScore + spanScore (clamped 0-10)
    9. Hint: "Maintain a regular publishing cadence — update sitemap lastmod dates to reflect content freshness"

    5. score = min(10, aiTxtScore + licenseSchemaScore)
    6. Hint: "Add /ai.txt with content licensing directives and CreativeWork schema with license property"
