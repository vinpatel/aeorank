  <action>
    Create packages/core/src/scanner/fetcher.ts:
    - createFetcher(config: ScanConfig): FetcherFn
    - Uses Node.js built-in fetch() with AbortSignal.timeout(config.timeout)
    - Sets User-Agent header to config.userAgent (default: "AEOrank/1.0 (+https://aeorank.dev)")
    - Wraps with p-limit(config.concurrency) for rate limiting (default: 3)
    - Handles Crawl-delay: if robotsInfo.crawlDelay > 0, add delay between requests using setTimeout
    - On 429/503: exponential backoff (1s, 2s, 4s) with max 3 retries
    - Returns { html: string, status: number, headers: Record<string, string>, responseTimeMs: number }
    - On error: returns { html: '', status: 0, headers: {}, responseTimeMs: 0 } with error logged

    Create packages/core/src/scanner/discovery.ts:
    - discoverUrls(startUrl: string, fetcher: FetcherFn, maxPages: number): Promise<string[]>
    - Step 1: Try to fetch and parse sitemap.xml at {origin}/sitemap.xml
    - Step 2: Fetch start page, extract internal links
    - Step 3: BFS crawl: queue discovered internal links, fetch and extract more
    - Deduplicate URLs (normalize before comparing)
    - Stop when maxPages reached
    - Only follow same-origin links
    - Skip common non-content paths: /wp-admin, /cdn-cgi, /api/, *.xml, *.json, *.pdf, *.zip, etc.

    Create packages/core/src/scanner/index.ts:
    - scanUrl(url: string, config?: Partial<ScanConfig>): Promise<ScanResult>
    - Merge config with DEFAULT_CONFIG
    - Step 1: Fetch robots.txt at {origin}/robots.txt, parse with parseRobotsTxt
    - Step 2: Check for existing llms.txt at {origin}/llms.txt
    - Step 3: Discover URLs (respecting Crawl-delay from robots.txt)
    - Step 4: Fetch and parse each discovered page
    - Step 5: Build ScanResult with pages, meta, timing
    - Score and grade set to 0 and 'F' (scorer fills these in Plan 03)
    - files set to [] (generators fill these in Plan 04)
    - Track total scan duration in milliseconds

    Update packages/core/src/index.ts to re-export scanUrl from scanner.

    - Test URL deduplication in discovery
