    - GeneratedFile: { name, content }
    - ScanConfig: { maxPages, concurrency, timeout, userAgent, respectCrawlDelay }
    - AeorankConfig: { site: { url, name, description }, output: { dir }, scanner: ScanConfig }

    Create packages/core/src/constants.ts:
    - DIMENSION_DEFS: Array of { id, name, weight, maxScore: 10 } for all 12 dimensions
      1. llms-txt (high), 2. schema-markup (high), 3. ai-crawler-access (medium),

    - AI_CRAWLERS: ["GPTBot", "ClaudeBot", "PerplexityBot", "Google-Extended", "anthropic-ai"]

    Create packages/core/src/utils.ts:
    - normalizeUrl(url): strip trailing slash, lowercase hostname
    - getGrade(score): lookup against GRADE_THRESHOLDS
    - getStatus(score): returns 'pass' | 'warn' | 'fail' based on STATUS_THRESHOLDS
    - calculateWeightedScore(dimensions): weighted sum using high=1.5, medium=1.0, low=0.5 multipliers, normalized to 0-100
    - slugify(text): lowercase, replace non-alnum with hyphens, collapse multiples, trim

    Create packages/core/src/index.ts re-exporting everything from types, constants, utils.
