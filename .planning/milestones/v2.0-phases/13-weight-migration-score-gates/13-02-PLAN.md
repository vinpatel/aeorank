**1. Add coherence gate to calculateAeoScore()** in scorer/index.ts:
After computing all dimension scores, look up the topic-coherence dimension result:
```typescript
const coherenceDim = dimensions.find(d => d.id === "topic-coherence");
const coherenceGated = coherenceDim && coherenceDim.score < 6;
// If gated, cap the final score at the coherence score * 10
// e.g., coherence=4 -> max score is 40
if (coherenceGated) {
  const cap = coherenceDim.score * 10;
  score = Math.min(score, cap);
}
```
This means a site with topic-coherence of 5/10 can never score above 50/100.

**2. Add duplication gate to scorePerPage()** in scorer/index.ts:
For each page, count duplicate paragraph blocks (same logic as scoreDuplicateContent: normalize paragraphs, count duplicates). If a page has 3 or more duplicate blocks, cap that page's score at 35:
```typescript
// After computing page score
const dupeCount = countDuplicateBlocks(page);
if (dupeCount >= 3) {
  score = Math.min(score, 35);
}
```
Add helper function `countDuplicateBlocks(page: ScannedPage): number` that normalizes paragraph text and counts repeated blocks (same algorithm used in scoreDuplicateContent).
